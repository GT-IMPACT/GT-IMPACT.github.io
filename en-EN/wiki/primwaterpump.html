Primitive Water Pump

Примитивная водокачка

Примитивная водокачка - это многоблочная машина, добавленная модификацией Impact Core.
Предназначена для генерации воды на начальных этапах и в трудодоступных местах без электропитания.

Primitive Water Pump в собранной форме

Машина генерирет воду без электричества в зависимости от биома в котором находится контролер.

Коэффициент генерации воды:


Коэффициент генерации воды - h. Коэффициент изменяется с помощью Output Hatch:
  • Pump Output Hatch: h = 1
  • ULV Output Hatch: h = 2
  • LV Output Hatch: h = 3

    • Биомный коэффициент:


    Биомный коэффициент - b. Коэффициент изменяется с помощью биома, в котором находится контролер:
  • Ocean, River: b = 1000 L/s
  • Swamplant: b = 800 L/s
  • Jungle: b = 350 L/s
  • Snow, Iceland: b = 300 L/s
  • Plains, Forest: b = 250 L/s
  • Taiga: b = 175 L/s
  • Beach: b = 170 L/s
  • Остальные: b = 100 L/s

    • Генерации воды:


    Вода генерируется при двух показателях (коэффициентах) по следующей формуле:
    Генерация воды (L/s) = h * b

    Пример генерации воды

    В биоме Plains c Pump Output Hatch генерация воды будет равна:
    Генерация воды = 250 * 1 = 250 L/s.

    В биоме River c LV Output Hatch генерация воды будет равна:
    Генерация воды = 1000 * 3 = 3000 L/s.

    Калькулятор генерации воды

    Для постройки этого мультиблока вам понадобятся 2 типа блоков: Primitive Pump Deck и Wood Frame Box.
    Output Hatch (устанавливается в красном квадрате на голограмме): Остальные люки обслуживания не требуются.

    *Не забудьте включить мультиблок с помощью Soft Mallet

    Для включения голограммы (подсказка при постройке) используйте Multiblock Machine Blueprint из мода TecTech, кликните им по контролеру мультиблока.
