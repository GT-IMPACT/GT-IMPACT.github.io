Parallel Super Computer

Супер Компьютер Параллельности

Супер компьютер позволяет раздавать параллельность для машин Impact.

Компьютер хранит параллельные поинты (Parallel Point, ПП), с возможностью их увеличения или расхода. Поинты нужны для раздачи параллельности машинам, каждая из которых ограничена по количеству параллельности: 4, 16, 64, 256.

Что такое параллельность? Параллельность - способность одновременно выполнять однотипные действия в несколько потоков.

Если рассматривать параллельность в игре, то это выполнение одного типа рецепта несколько раз за одно время (при этом затраты энергии пропорциональны количеству выполняемых рецептов).

Компьютер в собранной форме
(минимальная структура)
Для работы с компьютером, вам нужно подготовится:
- Установка Космического Спутника
- Установка Коммуникационной Вышки

Принцип работы:

После установки Коммуникационной вышки вы должны настроить частоту на которой будет работать вышка, компьютер и машины с помощью Ноутбука (Laptop).

Для регулирования поинтов параллельности существует Компьютерная Стойка (Computer Rack). Компьютерная Стойка имеет 4 слота, в которые помещаются Параллельные Процессоры (Parallel СPU). Существует 4 тира процессоров, которые вырабатывают: 4, 16, 64, 256 ПП.

Для передачи параллельности служат Параллельные Передатчики (Parallel Transmitter). Параллельные Передатчики существуют 4 тира, которые передают: 4, 16, 64, 256 ПП. При установке передатчика, в компьютере сразу будет зарезервировано нужное количество ПП для работы передатчика.

Чтобы передать ПП в нужную вам машину нужно установить в нее Параллельный Приемник (Parallel Receiver). Далее кликнуть по Параллельному Приемнику с помощью ноутбука, а затем кликнуть по Параллельному Передатчику на компьютере.
Если приемник/передатчик горит красным - ошибка подключения или подключение отсутсвует, если горит желтым - успешное подключение.

Важно!

Потребление энергии компьютера считается по следующей формуле:
Потребление (EU/t) = 8192 + Максимальное ПП.
Максимальное ПП - сумарное число ПП от установленных в компьютер Параллельных Процессоров.

Настройка передачи ПП с компьютера на машину

Для постройки этого мультиблока вам понадобятся блоки: Computer Casing и Empty Rack Casing.
  • Energy Hatch: 1x
  • Maintenance Hatch: 1x
  • Computer Rack: Nx
  • Parallel Transmitter: Nx
    • Остальные люки обслуживания не требуются.

    Важно!

    Структура компьютера может меняться (в длину), максимальная длина 16 блоков, минимальная - 2.

    Для включения голограммы (подсказка при постройке) используйте Multiblock Machine Blueprint из мода TecTech, кликните им по контроллеру мультиблока.
