INSTALL CLIENT

Installation only for licensed version of Minecraft

1. After downloading the client, you need to download MultiMC from the off site: multimc.org.


2. The archive will contain a MultiMC folder. Move it to the place you need and open MultiMC.


3. Select the language for the MultiMC interface and click next.


4. Select the required Java version. Set the minimum and maximum amount of random access memory (RAM)


5. Installation of MultiMC is complete. Click Finish to continue


6. Next, you need to connect your Minecraft account. Click on "Profiles", on "Account Management""


7. Then click "Add", in the window that opens, enter your data from your account Minecraft and click "OK"


8. Now you need to create a new modpack, for this click "Add Modpack"


9. In the window that opens, you need to write the name of the assembly and select "Import from zip", then click on the overview and specify the path to the downloaded archive with the assembly. Then click on "OK"


10. Done. Now you can start for the game, for this click "Start". Have a good game!
