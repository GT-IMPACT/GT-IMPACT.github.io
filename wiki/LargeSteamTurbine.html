Large Steam Turbine

Большая паровая турбина

Большая паровая турбина - это многоблочная машина-генератор, добавленная модификацией GregTech.
Турбина способна выработывать энергию, тратя при этом пар. Медленно разгоняется, используется для продолжительной выработки энергии.

Важно!

Изменена механика работы по сравнению с официальной версией GregTech, внимательно прочитайте представленую информацию.

Large Steam Turbine в собранной форме

Большая Паровая Турбина работает на пару GregTech и/или на пару из IndustrialCraft 2, выводя при этом дистилированную воду вдобавок к энергии.
Турбине требуется ротор для работы. Есть множество их вариантов, различающихся по размеру, материалу, эффективности работы и модификатору оптимальной подачи.

Важно!

Ротор является одноразовым, то есть вы вставляете его в интерфейс контролера турбины, турбина запоминает характеристики и удаляет ротор, характеристики сохраняются до тех пор, пока вы не сломаете контролер.

Эффективность:

Efficiency (Эффективность) - процентное отношение, влияющее на энергетический выход турбины.

Оптимальная подача:

Optimal Flow (Оптимальная подача) - это размер потока, необходимый для достижения оптимального энергетического выхода турбины. У каждого ротора есть индивидуальное значение оптимального потока.

Энергоплотность пара:

Fuel Value (Энергоплотность) пара = 0.5 EU/L

Генерация энергии:

Генерация энергии выполняется по формуле:
EU/t = (Optimal Flow / 20) * Fuel Values * (Efficiency / 100)

Пример генерации энергии из пара

Большой Ротор из материала Carbon с эффективностью - 120, с оптимальным потоком - 3000 L/s.

Генерация энергии:
EU/t = (3000 / 20) * 0.5 * (120 / 100) = 90 EU/t.

Калькулятор генерации энергии из пара

Для постройки этого мультиблока вам понадобятся блок: Turbine Casing.
  • Dynamo Hatch (Сзади по центру): 1x
  • Maintenance Hatch (Сбоку по центру): 1x
  • Input Hatch для пара (Сбоку по центру): 1x
  • Output Hatch для дистилированной воды (Сбоку по центру): 1x
    • Остальные люки обслуживания не требуются.

    Для включения голограммы (подсказка при постройке) используйте Multiblock Machine Blueprint из мода TecTech, кликните им по контролеру мультиблока.
