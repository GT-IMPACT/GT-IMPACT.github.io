Космический Спутник

Космический Спутник позволяет настроить беспроводную связь между механизмами на земле.

Космический Спутник в собранной форме

Для работы со Спутником, вам нужно подготовится:
- Установка Коммуникационной Вышки

Принцип работы:

После установки Космического Спутника вы должны настроить частоту с помощью Ноутбука (Laptop).

Для настройки частоты используется Коммуникационный Передатчик (Communication Transmitter).

Важно!

Для игры достаточно одного спутника.
Спутник не требует электрического питания.

Настройка частоты для Космического Спутника

Для постройки этого мультиблока вам понадобятся блоки: Space Satellite Casing.
  • Communication Transmitter: 1x
    • Остальные люки обслуживания не требуются.

    Важно!

    Спутник собирается только на орбите или (если отключена орбита в конфиге GalactiCraft) на луне.

    Для включения голограммы (подсказка при постройке) используйте Multiblock Machine Blueprint из мода TecTech, кликните им по контролеру мультиблока.
