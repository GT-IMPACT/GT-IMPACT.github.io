Saw Mill

Лесопилка

Лесопилка - это многоблочная машина, добавленная модификацией Impact Core.
Предназначена для обработки древесины, так как все остальные механики обработки дерева отключены (кроме ручного инструмента), вам понадобится эта машина для игрового процессинга.

Saw Mill в собранной форме

Машина имеет 3 режима работы. Режимы переключаются с помощью Screwdriver (Отвертка).

Режим 1. Planks & Sawdust (Доски и Опилки):


Данный режим позволяет из бревен получать доски и опилки, если у вас Output Bus ULV тира, вы получите на выходе только доски.

Режим 2. Wood Pulp & Sawdust (Древесная масса и Опилки):


Данный режим позволяет из бревен с водой получать древесная массу и опилки, если у вас Output Bus ULV тира, вы получите на выходе только древесную массу.

Режим 3. Only Sawdust (Только Опилки):


Данный режим позволяет из бревен получать только опилки.

Для постройки этого мультиблока вам понадобятся 2 типа блоков: Saw Mill Conveyor и Wooden Casing.
Люки обслуживания (устанавливаются в любом месте вместо Wooden Casing):
  • Energy hatch: 1x (Overclock - ускорение машины происходит только с MV тира)
  • Output Bus: до 1x
  • Input Hatches: до 1x
  • Input Bus: 1x
    • Остальные люки обслуживания не требуются.

    Для включения голограммы (подсказка при постройке) используйте Multiblock Machine Blueprint из мода TecTech, кликните им по контролеру мультиблока.
