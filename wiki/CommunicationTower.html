Communication Tower

Коммуникационная Вышка

Коммуникационная Вышка позволяет настроить беспроводную связь между спутником на орбите и механизмами на земле.

Коммуникационная Вышка
(свободная структура)

Для работы с вышкой, вам нужно подготовится:
- Установка Космического Спутника

Принцип работы:

После установки Космического Спутника вы должны настроить частоту с помощью Ноутбука (Laptop).

Для настройки подключения используется Коммуникационный Приемник (Communication Receiver). Частота должна совпадать с частотой спутника. Для установки связи нужно поключить все 4 приемника.

Важно!

У вышки есть радиус работы - 256 блоков, считается от контролера.
Несколько вышек могут быть подключены на одну частоту - частоту спутника.
Вышка не требует электрического питания.

Настройка частоты для подключения Коммуникационной Вышки к спутнику

Для постройки этого мультиблока вам понадобятся блоки: Tower Casing.
  • Communication Receiver: 4x
    • Остальные люки обслуживания не требуются.

    Важно!

    Вышка имеет уникальное свойство - свободная структура. Основа и приемники вышки являются обязательными.

    В голограмму встроена красивая модель вышки, которую можно построить. Для включения голограммы (подсказка при постройке) используйте Multiblock Machine Blueprint из мода TecTech, кликните им по контролеру мультиблока.
